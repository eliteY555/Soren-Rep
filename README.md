# 易诊通 - 智能在线问诊平台

## 项目介绍

易诊通是一款专注于智能医疗的在线问诊平台，旨在连接患者与医疗专家，提供便捷、高效、专业的诊疗服务。平台整合了人工智能辅助诊断与真实医师在线问诊，为用户提供全方位的健康医疗服务。

## 技术栈

### 前端技术栈
- **框架**：Vue 2.6.14
- **UI组件库**：Element UI 2.15.14
- **状态管理**：Vuex 3.6.2
- **路由管理**：Vue Router 3.5.1
- **HTTP客户端**：Axios 1.7.9
- **数据可视化**：ECharts 5.6.0
- **工具库**：
  - dayjs (时间处理)
  - crypto-js (加密处理)

### 后端技术栈
- **框架**：Spring Boot 3.5.0
- **ORM框架**：MyBatis
- **数据库**：MySQL
- **AI集成**：
  - LangChain4j (AI助手集成框架)
  - Ollama (本地AI模型)
  - 阿里云百炼平台 (云端AI服务)
- **文档处理**：Apache PDFBox
- **向量存储**：Pinecone
- **流式输出**：Spring WebFlux & Reactor

## 核心功能

### 1. 智能问诊系统
- 基于大语言模型的智能问诊助手"小易"
- 实时流式响应，提供自然的对话体验
- 专业医疗知识库支持，能够理解医学术语和辅助诊断

### 2. 医患沟通平台
- 患者与医生实时在线交流
- 病历共享与查看功能
- 诊断意见与处方开具

### 3. 电子病历管理
- 患者病历电子化存储
- 历史就诊记录查询
- 病情跟踪与分析

### 4. 专业医疗资源
- 多学科医师在线问诊
- 按专科、评价筛选医师
- 预约挂号系统

### 5. 数据统计与分析
- 就诊数据可视化展示
- 患者健康趋势分析
- 医生工作量与效率统计

## 服务人群

- **慢性病患者**：需要长期医疗指导的慢性病患者
- **亚健康人群**：有轻微不适但不便去医院的人群
- **健康管理需求人群**：对个人健康管理感兴趣的人群
- **偏远地区居民**：医疗资源不足地区的居民
- **忙碌白领**：没有时间去医院但需要健康咨询的职场人士

## 平台亮点

### 1. AI + 人工双重诊疗
结合AI智能初诊与专业医师复诊，提供高效准确的诊疗服务

### 2. 特色医疗服务
提供多样化的医疗服务，包括常见疾病诊断、健康咨询、用药指导等

### 3. 流式对话体验
采用最新的流式输出技术，实现类似ChatGPT的自然对话体验，提升用户体验

### 4. 完整病历管理
提供电子化病历管理，方便患者和医生追踪病情发展，制定长期健康方案

### 5. 数据安全保障
严格的数据加密和权限管理，保障患者隐私和医疗数据安全
