# 诊易通合作医院简介

## 重庆医科大学附属第一医院

**地址**：重庆市渝中区袁家岗友谊路1号  
**电话**：023-68811000  
**官网**：https://www.yd1y.com/  
**成立时间**：1957年  
**医院等级**：三级甲等  

**简介**：  
重庆医科大学附属第一医院创建于1957年，是一所集医疗、教学、科研、预防、保健、康复为一体的大型综合性三级甲等医院，也是国家区域医疗中心、国家临床重点专科建设项目单位。医院占地面积约20万平方米，建筑面积约40万平方米，开放床位3200余张。

**特色专科**：  
心血管内科、呼吸内科、内分泌科、神经内科、肝胆外科、骨科

**交通指南**：  
- **公交路线**：乘坐108路、121路、125路、139路、181路、209路、221路、261路、281路、467路、472路、483路等公交车至"重医附一院"站下车即可。
- **轨道交通**：乘坐轨道交通1号线或2号线至"袁家岗"站下车，从2号出口出站步行约500米可达。
- **自驾路线**：导航至"重庆医科大学附属第一医院"，院内设有地下停车场。
